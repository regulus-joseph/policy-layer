## Global Flags

- `--dev` — Dev mode (state in `~/.openclaw-dev`, shifted ports)
- `--profile <name>` — Named profile (state in `~/.openclaw-<name>`)
- `--container <name>` — Target container for execution
- `--no-color` — Disable ANSI colors
- `--json` — JSON output
- `--plain` — Plain text output (no styling)
- `-V, --version` — Print version
